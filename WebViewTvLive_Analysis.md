# WebViewTvLive 项目分析文档

## 1. 项目概述

**项目名称**: WebViewTvLive
**GitHub 地址**: https://github.com/hxh19950701/WebViewTvLive
**Star**: 4.7k
**Fork**: 550
**主要语言**: Kotlin (94.7%), JavaScript (5.3%)

## 2. 项目简介

这是一个使用**腾讯X5 WebView**开发的电视直播Android应用，允许用户通过嵌入网页的方式观看各大电视台的官方直播内容。

## 3. 核心技术

### 3.1 WebView 技术
- **腾讯X5 WebView**: 腾讯TBS浏览服务的内核，比系统原生WebView提供更好的兼容性
- 支持 m3u8 链接播放
- 基于 WebView 加载电视台官方直播页面

### 3.2 数据存储
- `data/` 目录下包含频道配置文件（`.dat` 格式）
- 目录结构:
  - `data/1/` - 央视和卫视频道
  - `data/2/` - 省级频道
  - `data/3/` - 地方频道
  - `data/4/` - 其他频道

### 3.3 配置文件类型
- `full.dat` - 完整频道列表
- `cctv_and_guangdong_tv_single.dat` - 央视+广东卫视
- `cctv_and_hunan_tv_single.dat` - 央视+湖南卫视
- `cctv_and_satellite_tv_single.dat` - 央视+卫星电视
- `china_mainland.dat` - 中国大陆频道
- `server_config.dat` - 服务器配置

## 4. 主要功能

1. **自动更新频道列表** - 支持频道列表的自动更新
2. **多直播源** - 同一频道可能有多个直播源
3. **应用自定义设置** - 用户可自定义设置
4. **多平台适配** - 手机、平板、电视、车机
5. **数字键换台** - 遥控器数字键换台
6. **上下键换台反转** - 可自定义换台方向
7. **TBS 内核可选** - 可选择是否启用腾讯TBS内核

## 5. 技术特点

### 优势
- 直接使用官网直播链接，非常稳定可靠
- 兼容 m3u8 链接格式
- 利用 WebView 加载直播页面，绕过直接流媒体解析

### 局限性
- 基于 WebView，对设备性能有一定要求
- 载入频道需要等待时间
- 无法选择视频清晰度，帧率有限（大部分24FPS）
- 受限于官网是否有直播

## 6. 直播协议类型

根据项目描述，支持以下协议：

| 协议类型 | 说明 |
|---------|------|
| **m3u8** | HLS流媒体协议，广泛用于直播 |
| **WebView加载** | 通过WebView加载官网直播页面 |
| **HTTP/HTTPS** | 常规HTTP流 |

## 7. 数据文件格式

项目的数据文件（`.dat`）是经过处理的二进制/加密格式，无法直接读取。原始直播URL存储在加密数据中，通过应用内置逻辑解密后加载。

## 8. 相关技术栈

- **前端**: JavaScript (用于WebView内嵌页面)
- **后端**: Kotlin (Android原生开发)
- **构建工具**: Gradle
- **WebView**: 腾讯X5 WebView (TBS浏览服务)
